## Lesson 2.1 — Search and buy

1. Go to **spaceship.com** (or your registrar of choice)
2. Search your chosen name
3. Add to cart — enable **auto-renew**
4. Turn on **WHOIS privacy** if included

**Checklist**
- [ ] Domain purchased
- [ ] Receipt saved
- [ ] Login credentials in your password manager

## Lesson 2.2 — Nameservers to Cloudflare

You will host DNS in Cloudflare (next module). In Spaceship:

1. Domain management → **Nameservers**
2. Replace defaults with Cloudflare's two NS records (you get these when you add the site in Cloudflare)

**Checklist**
- [ ] Nameservers updated (propagation can take up to 24h; usually faster)
- [ ] Old parking page no longer matters — Cloudflare is in control

## Lesson 2.3 — Protect the brand

**Checklist**
- [ ] Secure the same handle on Instagram/YouTube if you plan to post builds
