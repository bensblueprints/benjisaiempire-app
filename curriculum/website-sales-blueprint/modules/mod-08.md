## Lesson 8.1 — Pipeline order

**Watch:** [Github Sign Up Where Your Code Lives](https://www.youtube.com/watch?v=pBxTJ6rb1GY)

1. GitHub repo
2. Blank deploy to Netlify (pipeline wired)
3. Scrape/audit site
4. Design + build (Framer Motion, components)
5. Push → auto deploy
6. `REDESIGN-REPORT.md` for the demo

## Lesson 8.2 — Booking backends

Only build scheduling/booking if the client actually takes appointments. Otherwise skip — speed wins.

## Lesson 8.3 — Quality bar for demo

Minimum before **Site demoed**:
- [ ] Mobile layout not broken
- [ ] Phone + address visible
- [ ] One clear CTA (Call / Book / Directions)
- [ ] Loads in under 4s on LTE
