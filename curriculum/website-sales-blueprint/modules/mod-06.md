## Lesson 6.1 — Create the pipeline

Pipeline name: **Website Sales**

Stages (exact order):
1. **Agreed to free site**
2. **Free site created**
3. **Site demoed**
4. **Invoice sent**
5. **Invoice paid**

**Checklist**
- [ ] Pipeline created in GHL
- [ ] No extra stages (complexity kills speed)

## Lesson 6.2 — What moves the deal

| Stage | Meaning |
|-------|---------|
| Agreed to free site | Verbal yes + URL captured + build scheduled |
| Free site created | Redesign deployed at preview URL |
| Site demoed | They saw it live or on screen share |
| Invoice sent | $500 + hosting terms sent |
| Invoice paid | Money hit — start DNS/handoff |

## Lesson 6.3 — Morning pipeline review

Every morning before dialing:
- [ ] Open **Website Sales**
- [ ] Anything stuck >3 days gets a follow-up call
- [ ] Today's goal: X new **Agreed to free site** + move demos/invoices
